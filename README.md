# Material DS Simulation

Material DS Simulation is a simulation of a DS-aware AI code generation pipeline using Material Design 3, MUI, React + TypeScript.

## What's inside
- Theme tokens
- DS component library
- Screens
- Storybook stories
- Design system context file: `design-system.md`

## How to run
```bash
npm run dev
```

```bash
npm run storybook
```

## Tech stack
- React
- TypeScript
- Vite
- MUI v5
- Storybook

## Purpose
Proving that structured DS context improves AI-generated code quality.
