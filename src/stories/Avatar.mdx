# Avatar

Avatar represents a person, account, or identity placeholder through generic, monogram, check, and three-dimensional variants. It solves the need to show compact identity markers in a consistent shape.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use avatars to represent people, accounts, or identity placeholders',
    'Use monograms when no image-based identity is available',
    'Keep avatar labels short and scannable',
    'Use the three-dimensional variant only when that visual style is intended',
    'Keep avatar usage consistent within the same interface area',
  ]}
  donts={[
    'Use avatars as general-purpose status icons',
    'Mix too many avatar styles without hierarchy',
    'Put long labels inside the avatar',
    'Use check avatars when identity is still required',
    'Rely on color alone to communicate meaning',
  ]}
/>
