# Divider

Divider separates related content into readable sections without becoming a focal element. It works best as quiet structure inside lists, forms, sheets, and stacked layouts.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use dividers to support content grouping',
    'Choose inset styles when rows contain leading elements',
    'Use labeled dividers only for true section breaks',
    'Keep divider treatment subtle',
    'Prefer spacing first, then divider when separation still needs help',
  ]}
  donts={[
    'Use dividers as decoration',
    'Over-separate every row in a dense layout',
    'Add long descriptive copy as a divider label',
    'Use vertical dividers where spacing alone is enough',
    'Rely on divider color as the only hierarchy cue',
  ]}
