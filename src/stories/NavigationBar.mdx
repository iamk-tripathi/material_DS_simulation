# Navigation Bar

Navigation Bar presents a small set of destinations for moving between primary sections. It adapts the same destination model across bottom bar, rail, and drawer presentations so navigation can match the available space.

## Variants

## Do's and Don'ts

<DosDonts
  dos={[
    'Keep destination count between two and five',
    'Use short clear labels that map to primary app sections',
    'Choose bar, rail, or drawer based on available space and navigation prominence',
    'Keep the selected destination clearly visible',
    'Use drawer only when the layout can support a persistent side region',
  ]}
  donts={[
    'Cram more than five primary destinations into the pattern',
    'Use vague labels that overlap in meaning',
    'Mix top-level and secondary destinations together',
    'Change presentation type without a layout reason',
    'Rely on icons alone when labels are needed for clarity',
  ]}
/>
