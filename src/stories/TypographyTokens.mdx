<DocSection title="Usage Rules" description="Recommended UI usage for each level of the Material Design 3 type scale.">
  <ul>
    <li>Display: hero text, large empty states, and the strongest expressive moments.</li>
    <li>Headline: page sections, feature headings, and primary content grouping.</li>
    <li>Title: app bars, card titles, sheet titles, and compact heading surfaces.</li>
    <li>Label: buttons, chips, tabs, toggles, and compact control text.</li>
    <li>Body: paragraph text, supporting text, metadata, and longer-form reading.</li>
  </ul>
</DocSection>
