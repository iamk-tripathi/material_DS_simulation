# Menu

Menu presents a transient list of contextual actions using baseline, standard, and vibrant treatments. It solves the need to show grouped action choices in a compact overlay surface.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use menus for contextual action lists',
    'Keep item labels concise and scannable',
    'Group menu items only when grouping improves clarity',
    'Use vibrant menus intentionally when stronger emphasis is needed',
    'Preserve active and disabled feedback where relevant',
  ]}
  donts={[
    'Use menus for long-form navigation',
    'Overgroup menu items without a clear reason',
    'Hide disabled or active state feedback',
    'Use vibrant menus everywhere by default',
    'Overload a single menu with unrelated actions',
  ]}
/>
