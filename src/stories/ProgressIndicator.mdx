# Progress Indicator

Progress Indicator communicates ongoing status, completion, or background loading. It helps users understand whether work is measurable, still in flight, or waiting on an unknown duration.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use determinate progress only when the value is actually known',
    'Use indeterminate indicators for unknown duration tasks',
    'Choose circular or linear form based on available space',
    'Anchor loading states inside a container when context matters',
    'Match thickness and style to the emphasis of the surface',
  ]}
  donts={[
    'Show determinate values that are not trustworthy',
    'Use multiple competing loading indicators in one region',
    'Rely on motion alone to explain status',
    'Use a heavy loading treatment for very minor background tasks',
    'Leave users without context when loading takes time',
  ]}
/>
