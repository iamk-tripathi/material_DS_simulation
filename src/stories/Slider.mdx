# Slider

Slider adjusts a value across a constrained range through direct manipulation. It supports single-value, centered, and range-based input without switching component families.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use standard sliders for single continuous values',
    'Use centered sliders when the midpoint matters semantically',
    'Use range sliders for lower and upper bounds',
    'Choose slider size based on emphasis and touch comfort',
    'Provide labels or context so the value meaning is clear',
  ]}
  donts={[
    'Use a slider when only a few discrete choices exist',
    'Use two separate standard sliders instead of a true range slider',
    'Rely on color alone to explain the current value',
    'Make the drag area too small for the intended device',
    'Disable a slider without explaining why the value is unavailable',
  ]}
/>

## Accessibility

<DocSection title="Value Context">

</DocSection>
