# Carousel

Carousel presents browsable content collections across mobile and tablet contexts. It solves the need to surface multiple related items in a horizontally scrollable format with different layout densities.

## Do's and Don'ts

<DosDonts
  dos={[
    'Choose the carousel layout based on browsing density and context',
    'Match mobile and tablet contexts to the intended item count',
    'Use aspect ratio intentionally to support the content type',
    'Keep card content concise enough for horizontal browsing',
    'Use full-screen layout only when stronger immersion is needed',
  ]}
  donts={[
    'Use the same density for every context by default',
    'Overfill carousel cards with dense text',
    'Mix unrelated aspect ratios without a content reason',
    'Use full-screen layout when standard browsing is enough',
    'Treat carousel as a replacement for long-form list content',
  ]}
/>
