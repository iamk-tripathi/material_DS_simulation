# Switch

Switch toggles an immediate on or off setting. It is the right choice when a preference takes effect as soon as the control changes, without requiring an extra confirmation step.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use switch for immediate binary settings',
    'Pair each switch with a clear text label',
    'Keep visible focus treatment intact',
    'Use icon support only when it clarifies state meaning',
    'Explain disabled settings when users might expect them to be editable',
  ]}
  donts={[
    'Use switch for a choice that still needs submission',
    'Use switch in place of radio buttons for exclusive options',
    'Decorate the switch with unrelated icons',
    'Rely on color alone to explain on and off',
    'Hide the effect of a setting behind unclear wording',
  ]}
/>
