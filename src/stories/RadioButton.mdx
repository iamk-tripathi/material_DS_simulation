# Radio Button

Radio Button represents one choice inside a mutually exclusive set. Use it when the user must choose exactly one option from a group and should be able to compare all options before deciding.

## Do's and Don'ts

<DosDonts
  dos={[
    'Use radio buttons for mutually exclusive choices',
    'Keep options in one clearly labeled group',
    'Preserve visible focus treatment for keyboard use',
    'Keep labels short and easy to compare',
    'Use disabled state only when the choice may become available later',
  ]}
  donts={[
    'Use radio buttons for multi-select behavior',
    'Split related radio options across distant layouts',
    'Remove the visible selected indicator',
    'Use color alone to communicate the chosen option',
    'Hide the group label from users',
  ]}
