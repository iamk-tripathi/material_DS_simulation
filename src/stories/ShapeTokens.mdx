<DocSection title="Usage Rules" description="Material Design 3 guidance for matching shape scale to component type.">
  <p><strong>Source: Material Design 3 specification</strong></p>
  <ul>
    <li>None (<code>0dp</code>) -&gt; No rounding. Used for full-bleed surfaces.</li>
    <li>Extra-small (<code>4dp</code>) -&gt; Chips, text fields, small components.</li>
    <li>Small (<code>8dp</code>) -&gt; Cards (small), menus, snackbars.</li>
    <li>Medium (<code>12dp</code>) -&gt; Cards (default), FAB (small).</li>
    <li>Large (<code>16dp</code>) -&gt; Bottom sheets, navigation drawers, extended FAB.</li>
    <li>Extra-large (<code>28dp</code>) -&gt; Dialogs, large FAB.</li>
    <li>Extra-extra-large (<code>48dp</code>) -&gt; Bottom sheets (large).</li>
    <li>Full (<code>50%</code>) -&gt; Badges, sliders, icon buttons, search bars.</li>
  </ul>
</DocSection>
