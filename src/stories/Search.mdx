# Search

Search combines query entry with contextual surface treatments and optional identity cues. It supports compact bar search, docked search modules, and broader full-screen search presentations.

## Do's and Don'ts

<DosDonts
  dos={[
    'Choose bar, docked, or full-screen search based on task prominence',
    'Use supporting text only when it adds clear value',
    'Keep the field easy to identify as editable',
    'Use avatar-leading search only when identity context matters',
    'Provide search context near the field when scope may be unclear',
  ]}
  donts={[
    'Rely on placeholder text as the only accessible label',
    'Use full-screen search for a minor utility action',
    'Add supporting text that repeats the placeholder',
    'Overload the leading area with too many competing affordances',
    'Hide search scope when it affects results meaningfully',
  ]}
/>
