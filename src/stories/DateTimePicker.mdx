# Date Time Picker

Date Time Picker collects temporal values through modal, input, docked, dial, and keyboard-driven formats. It solves the need to enter dates and time-related values in a structured flow.

## Do's and Don'ts

<DosDonts
  dos={[
    'Choose the picker variant based on the input flow and device context',
    'Use range mode only when two related temporal values are required',
    'Match the time format to the product context',
    'Keep date entry labels clear and unambiguous',
    'Use docked layouts only when the surrounding UI can support them',
  ]}
  donts={[
    'Mix picker modes without a clear reason',
    'Use range mode when only a single value is needed',
    'Hide the time format when the picker depends on it',
    'Overcrowd the picker surface with unrelated controls',
    'Rely on layout alone to explain the picker mode',
  ]}
/>
