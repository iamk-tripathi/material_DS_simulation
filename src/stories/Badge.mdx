# Badge

Badge is used to indicate a notification, item count, or related status on a navigation destination. It solves the need to show a compact status indicator attached to a navigation destination.

## Variants

<DocSection title="Large" description="Use when the badge needs to show a readable count or short status value.">
  <ComponentPreview caption="Large Badge" note="Use for compact counts that need to remain visible on a navigation destination.">
    <ThemeProvider theme={theme}>
      <DSBadge size="large" label="3" />
    </ThemeProvider>
  </ComponentPreview>
</DocSection>

<DocSection title="Small" description="Use when a status indicator is needed without any text value.">
  <ComponentPreview caption="Small Badge" note="Use as a dot indicator when a count is not required.">
    <ThemeProvider theme={theme}>
      <DSBadge size="small" />
    </ThemeProvider>
  </ComponentPreview>
</DocSection>

## States

<DocSection title="Overflow" description="Use when the count exceeds the maximum visible value.">
  <ComponentPreview caption="Overflow Badge" note="Values above the maximum collapse to the max-plus format.">
    <ThemeProvider theme={theme}>
      <DSBadge size="large" label="100" max={99} />
    </ThemeProvider>
  </ComponentPreview>
</DocSection>

<DocSection title="No Label" description="Use when the large badge container is needed without a visible count.">
  <ComponentPreview caption="Large Badge Without Label" note="Only use when the large badge shape itself communicates the required status.">
    <ThemeProvider theme={theme}>
      <DSBadge size="large" />
    </ThemeProvider>
  </ComponentPreview>
</DocSection>

## Do's and Don'ts

<DosDonts
  dos={[
    'Use badge counts for notifications or related status only',
    'Use the small badge when text is not needed',
    'Keep large badge labels short and compact',
    'Use error tokens consistently for the badge surface',
    'Use max values to prevent unreadable counts',
  ]}
  donts={[
    'Use badges as standalone labels without an attached destination',
    'Place long text strings inside a large badge',
    'Use the small badge to show readable counts',
    'Change badge colors away from the error token set',
    'Rely on badge color alone for accessibility context',
  ]}
/>
