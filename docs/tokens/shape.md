# Shape Tokens

## Overview
This page documents the visible shape scale values and corner radius values provided in the material.

## Shape Scale

| Token name | dp value | Corner radius value |
|---|---|---|
| `None` | `0dp` | `0` |
| `Extra-small` | `4dp` | `4` |
| `Small` | `8dp` | `8` |
| `Medium` | `12dp` | `12` |
| `Large` | `16dp` | `16` |
| `Large-increased` | `20dp` | `20` |
| `Extra-large` | `28dp` | `28` |
| `Extra-large-increased` | `32dp` | `32` |
| `Extra-extra-large` | `48dp` | `48` |
| `Full` | `50%` | `1000` |

## Usage
- Which scale value maps to which component size: This information is not visible in the provided material. Please confirm.
